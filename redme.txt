== Description ==

Howcan Pro Addons extends the free **Howcan** WordPress theme by adding advanced customization options, layouts, color schemes, and header/footer controls.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/howcan-pro-addons/` directory, or install through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Enjoy new premium options in the Howcan theme customizer.
